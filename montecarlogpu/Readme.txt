Here's how to make this works:

Basic install
------------
- Download the latest version of Aparapi (http://code.google.com/p/aparapi/downloads/list)
- Unzip it somewhere
- Set APARAPI_HOME variable to Aparapi directory
- In Eclipse, add a variable named APARAPI_HOME to the same directory
- You can now import the projet in Eclipse

Ant
------------
- Call ant in this directory
- It should work
